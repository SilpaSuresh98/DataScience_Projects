This project focuses on analyzing the employee performance data of INX Future Inc., a global data analytics and automation company. The goal is to identify the core factors affecting employee performance and build a predictive model that can support hiring and HR decision-making.

**Project Objective:**

The key objectives defined by IABAC for this project are:

- Develop a machine learning model that predicts employee performance ratings based on various input features.

- Provide actionable recommendations to improve employee performance based on data insights.
